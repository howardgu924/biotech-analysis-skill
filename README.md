# biotech-analysis-skill

An experimental Claude Code skill for biotech, pharmaceutical, diagnostics, and life-science tools investment research.

This skill helps structure research on companies such as OnKure, GRAIL, Hengrui Medicine, and similar biotech/pharma businesses. It focuses on clinical catalysts, pipeline quality, mechanism of action, cash runway, reimbursement, competition, and rNPV-style valuation.

## Use Cases

- Analyze a pre-revenue oncology biotech
- Review a diagnostics company
- Compare competing drug candidates
- Build a catalyst timeline
- Estimate cash runway and dilution risk
- Create a bull/base/bear valuation framework
- Review China pharma companies with NRDL/VBP and global licensing considerations
